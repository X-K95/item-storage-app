# 物品收纳助手APP

一款帮助用户管理、归类和查找物品的智能应用，支持语音播报物品位置和灯光提示功能。

## 核心功能

### 1. 物品管理
- **物品录入**：支持手动输入和拍照识别两种方式
- **智能分类**：按类别、房间、存储位置多层级分类
- **标签系统**：自定义标签（常用、季节性、贵重物品等）
- **图片上传**：为物品添加照片，便于快速识别

### 2. 智能查找
- **语音搜索**：支持自然语言查询（如“我的护照在哪里？”）
- **关键词搜索**：按名称、类别、位置等多维度搜索
- **图像识别搜索**：上传物品照片快速定位

### 3. 语音播报与灯光提示
- **位置语音播报**：查找到物品后，自动语音播报详细位置
- **灯光联动**：通过智能家居设备控制物品所在区域的灯光亮起
- **多设备同步**：支持手机、智能音箱、智能手表等多设备接收提醒

## 技术实现

### 前端技术栈
- **React Native**：跨平台移动应用开发框架
- **Expo**：快速开发和部署工具
- **Audio API**：语音播报功能
- **Image Picker**：图片选择和上传

### 后端技术栈
- **Node.js**：服务器端开发
- **MongoDB**：数据库存储
- **Express**：API接口开发

### 智能家居集成
- **米家平台**：小米智能家居设备控制
- **HomeKit**：苹果智能家居平台
- **Google Home**：谷歌智能家居平台

## 安装与运行

### 环境要求
- Node.js 14.0 或更高版本
- Expo CLI 4.0 或更高版本
- iOS 10.0 或更高版本 / Android 5.0 或更高版本

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/your-repo/item-storage-app.git
cd item-storage-app
```

2. **安装依赖**
```bash
npm install
```

3. **配置API密钥**
- 在 `item_storage_app.js` 中替换 `YOUR_API_KEY` 为你的语音合成API密钥
- 推荐使用以下语音合成服务：
  - 科大讯飞语音合成
  - 百度语音合成
  - 腾讯云语音合成

4. **运行项目**
```bash
expo start
```

5. **在设备上运行**
- iOS：使用Xcode模拟器或Expo客户端
- Android：使用Android Studio模拟器或Expo客户端

## 使用说明

### 添加物品
1. 点击“+ 添加物品”按钮
2. 填写物品名称、类别和位置
3. 可选：上传物品照片
4. 点击“添加”按钮保存

### 查找物品
1. 在搜索框中输入关键词
2. 点击搜索结果中的“🔊”按钮
3. 系统将语音播报物品位置
4. 同时触发所在区域的灯光提示

### 智能家居设置
1. 在APP中添加智能家居设备
2. 配置灯光联动规则
3. 测试灯光提示功能
